## Current Portfolio / Desktop

- Width target: 1440 desktop frame, with content containers at 1200px and wide visual sections up to 1720px.
- Background: cream/off-white site base with dark navy command-center hero and game-direction bands.
- Header: sticky dark/navy transparent shell, desktop nav links, mobile menu at small widths.
- Hero: left-side identity/copy/CTA stack plus a dark product-command visual containing artifact preview cards and journey logic.
- Middle narrative: game-direction dark band, about/profile section, organization/tool marquees, mindset/process/skills bands, proof snapshot.
- Work evidence: professional Vega projects with image samples, additional experience, academic projects.
- Closing: value pillars, supporting work sample, contact CTA, footer.

## Section Order

1. home - Hero: Hero
2. game-direction - GameThinking: Game direction / product-domain positioning
3. about - About: Profile, background, education
4. organizations - OrganizationsStrip: Credibility logo marquee

7. snapshot - ProofSnapshot: Stats and recommendation/social proof cards
8. experience - ProfessionalExperience: Vega projects and additional professional entries
9. additional-experience - AdditionalExperience: Supplemental operating/data background
10. projects - AcademicProjects: Academic and portfolio project cards

14. footer - Footer: Footer contact and focus note
